## 🚀 Sobre o Projeto

Um portal de notícias moderno e responsivo com visual dark, categorias de tecnologia e seção de notícias mais lidas. O layout foi construído inteiramente com **HTML** e **CSS**, sem o uso de frameworks externos.

## ✨ Funcionalidades

- Navbar com menu de categorias (IA, Blockchain, Hologramas, Internet, Vestíveis, Realidade Aumentada e Virtual)
- Hero com destaque para notícia principal e cards secundários
- Seção **"Mais lidas da semana"** com cards por categoria
- Layout responsivo com grid e flexbox
- Tema dark com paleta de cores consistente

## 🛠️ Tecnologias

- **HTML5** — estrutura semântica da página
- **CSS3** — estilização, Grid Layout e Flexbox
